# Rosasite-Jewelry
# 💎 Bosasite Jewelry – Luxury Responsive E-Commerce Website

Bosasite Jewelry is a modern, elegant, and fully responsive jewelry website designed using **HTML5, CSS3, JavaScript, and Bootstrap 5**. Inspired by luxury jewelry brands, this project delivers a premium shopping experience with a clean UI, smooth animations, and mobile-friendly design.

## ✨ Features

* 💎 Luxury & Elegant User Interface
* 📱 Fully Responsive Design
* 🛍️ Product Collection Section
* 💍 Jewelry Categories
* ⭐ Featured Products
* ❤️ Wishlist & Shopping Cart UI
* 🔍 Search Icon
* 📩 Newsletter Subscription
* 📞 Contact Section with Google Maps
* 🎨 Modern Color Palette Inspired by the Brand Logo
* ⚡ Smooth Animations & Hover Effects
* 🧭 Sticky Navigation Bar
* 🖥️ Cross-Browser Compatible

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Bootstrap Icons
* Google Fonts

This project demonstrates modern front-end development skills, responsive web design, and clean UI/UX practices, making it an excellent addition to a web development portfolio.
